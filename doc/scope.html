Scope

Purpose

The purpose of this project is to demonstrate understanding of concepts learned during the course of pursuing a degree in Information Technology, with a focus on specific courses and their curricula. The focus areas of this project include data modeling, database design, application development, system development, user interface design, transactional processing, system integration, and data protection. This project also serves an additional purpose to demonstrate the ability to making voting easier for citizens.

Objective

The objective of this project is to provide a secure, accessible, internet-enabled application that is built on a current technology standards platform for the purpose of providing voter registration and management, ballot management, and voting. This system will provide functionality with most current browsers on devices with an internet connection, and will also provide features for offline use and manual registration for traditional, non-computer users. This system will expand methods for voting by creating a system for voting activities that can operate without the need for polling workers, temporary facilities, or specific operating hours.

High-Level Requirements (Scope)

This project will consist of multiple browser-based applications. The Voter Registration Application will provide the ability for a new or existing voter to login, register to vote, or update their existing registration, and logout. The Ballot Casting Application will provide the ability for an existing voter to login, view available ballots, cast a ballot, and logout. The Administration Application will provide the ability for designated users to administer voter registrations, enter manual registrations, create and configure ballots, and administer ballot casting by opening or closing voting windows, and tabulating ballots cast.

These applications will include a data model suitable for deployment to a RDBMS that provides encryption of data at rest and in transit, robust scalability, and transactional activity such as rollback. These applications will be designed to be deployed on a standard J2EE application server that supports secure, encrypted communications, scalability, user authentication and authorization models, and web service deployment. This project will be designed to support external service integration through web services, allowing easy integration to an enterprise service bus, for features such as address validation. This project is expected to be completed within 180 hours.
