Data Model

The data model, explained in a navigable HTML file.

Sources

The schema artifacts are created and populated via several SQL script files. You can obtain these scripts from GitHub as they are prevented from being served over HTTP from this site for security.

  • DDL script for initial database entity/relation creation: voice-ddl.sql
  • DML script to create stored procedures, functions, triggers: voice-dml.sql
  • Data script for initial population of data for system: voice-data.sql
  • Testing script to verify function of DML and process flows: voice-test.sql
